# 📅 AgendaOne - Gestion d'événement en ligne

## 📝 Description
Ce projet est une application web de productivité développée en PHP permettant de gérer son emploi du temps en ligne. L'objectif est de mettre en place une interface moderne et élégante pour planifier, organiser et suivre ses événements personnels ou professionnels de manière simple et intuitive.

## 🚀 Fonctionnalités
- **Mon Agenda** : Visualisation claire et structurée de l'emploi du temps et des événements programmés.
- **Créer un événement** : Formulaire dynamique pour ajouter de nouveaux événements ou bloquer des créneaux.
- **Espace Client** : Système d'inscription et de connexion sécurisé.
- **Interface Responsive** : Design épuré et adapté à tous les types d'écrans (Mobiles, Tablettes, Ordinateurs).

## 🛠️ Technologies utilisées
- **Backend** : PHP
- **Frontend** : HTML5, CSS3, JavaScript
- **Serveur local** : Laragon
- **Gestion de version** : Git & GitHub
